## Quick Tour: How Easy Is It?

```bash
# Your existing Docker app
docker run -p 8080:80 nginx

# The same app in a TEE with dstack
dstack deploy nginx -p 8080:80
```

That's it. Your app now runs in a hardware-isolated environment with:

* Memory encryption
* Remote attestation
* Cryptographic proof of execution
* Certificate management
* Secure key storage
