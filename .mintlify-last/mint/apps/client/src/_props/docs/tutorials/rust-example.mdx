## What You'll Learn

Once completed, this tutorial will cover:

* Building a high-performance Rust web service with Actix-web
* Creating optimized Docker images for Rust
* Deploying to dstack TEE with minimal overhead
* Implementing secure, type-safe endpoints
* Zero-copy attestation handling

## Prerequisites

* Rust toolchain installed
* Basic knowledge of Actix-web or similar frameworks
* Docker basics
* dstack installed and running
