# Key Features

dstack provides a comprehensive set of features designed to make secure computing accessible, manageable, and scalable. Each feature is built with both developers and enterprises in mind, ensuring ease of use without compromising security.

## Core Security Features

### üîí **Seamless TEE Deployment**

Transform any Docker container into a secure Confidential VM with zero code changes.

**Example:**
```yaml

  app:
    image: myapp:latest
    ports:
      - "8080:8080"
```

### üõ°Ô∏è **Remote Attestation**

Cryptographically verify that your application is running unmodified in a genuine TEE.

**Features:**
- Hardware-signed attestation quotes
- Verifiable measurements of code and configuration
- Integration with Intel DCAP for quote verification
- Custom event log for application-specific attestation

**API Example:**
```javascript
const { DstackClient } = require('@phala/dstack-sdk');
const client = new DstackClient();

// Get attestation quote
const quote = await client.getQuote();
console.log('TEE Quote:', quote);
```
