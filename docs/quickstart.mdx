# Quick Start Guide

This guide will help you get started with dstack quickly. Follow these steps to set up your environment and create your first project.

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js (v14 or later)
- npm or yarn
- Git

## Installation

Install dstack using npm:

```bash
npm install -g dstack-cli
```

Or using yarn:

```bash
yarn global add dstack-cli
```

## Create Your First Project

1. Initialize a new dstack project:

```bash
dstack init my-first-project
cd my-first-project
```

2. Configure your project:

```bash
dstack configure
```

3. Deploy your project:

```bash
dstack deploy
```
