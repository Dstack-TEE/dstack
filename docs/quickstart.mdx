## 🚀 5-Minute Quick Start

### Prerequisites

- Ubuntu 24.04 with Intel TDX support
- 16GB RAM, 100GB storage
- Basic familiarity with Docker

### Step 1: Install Dependencies

```bash
# Install system packages
sudo apt update && sudo apt install -y \
    build-essential chrpath diffstat lz4 \
    wireguard-tools xorriso git curl

# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
```

### Step 2: Clone and Build

```bash
# Clone repository
git clone https://github.com/Dstack-TEE/meta-dstack.git --recursive
cd meta-dstack

# Configure and build
./build.sh hostcfg
./build.sh build

# Download guest image
./build.sh dl 0.5.1
```

### Step 3: Start Services

```bash
# Start in separate terminals
./dstack-kms -c kms.toml
sudo ./dstack-gateway -c gateway.toml
./dstack-vmm -c vmm.toml
```

### Step 4: Deploy Your First App

Create a simple `docker-compose.yml`:

```yaml
version: '3.8'
services:
  app:
    image: nginx:alpine
    ports:
      - "80:80"
```

Deploy it:
```bash
# Via Web UI
# Navigate to http://localhost:9080

# Or via CLI
dstack deploy -f docker-compose.yml
```

## 🎯 Common Use Cases

### Deploy a Python Web App

```python
# app.py
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello from dstack!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
```

```dockerfile
# Dockerfile
FROM python:3.11-slim
WORKDIR /app
RUN pip install flask
COPY app.py .
CMD ["python", "app.py"]
```

## 🛠️ Essential Commands

```bash
# Check installation
dstack version

# List deployments
dstack list

# View logs
dstack logs <app-name>

# Get attestation
dstack attest <app-name>
```

## 🔧 Quick Configuration

### Minimal Configuration Files

<details>
<summary>kms.toml</summary>

```toml
[general]
log_level = "info"

[rpc]
listen = "127.0.0.1:8000"
```

</details>

<details>
<summary>gateway.toml</summary>

```toml
[network]
public_ip = "YOUR_IP"
srv_domain = "app.example.com"

[gateway]
listen = "0.0.0.0:443"
```

</details>

<details>
<summary>vmm.toml</summary>

```toml
[resources]
cid_start = 33000
max_vms = 10

[api]
listen = "127.0.0.1:9080"
```

</details>

## 🆘 Quick Troubleshooting

### Services Won't Start
```bash
# Check ports
sudo netstat -tulpn | grep -E '9080|443'

# View logs
sudo journalctl -u dstack-vmm -n 50
```

### Deployment Fails
```bash
# Check Docker image
docker-compose config

# Verify resources
dstack resources
```

### Can't Access App
```bash
# Check status
dstack status <app-name>

# Test connectivity
curl http://localhost:9080/api/health
```
