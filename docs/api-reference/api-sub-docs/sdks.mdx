# Official SDKs

## JavaScript/TypeScript SDK

**Install:**
```bash
npm install @phala/dstack-sdk
```

**Features:**
- Viem integration for Ethereum
- Solana Web3.js integration
- Environment variable encryption utilities

---

## Python SDK

**Install:**
```bash
pip install dstack-sdk
```

**Blockchain Integration:**
- Ethereum: `pip install "dstack-sdk[eth]"`
- Solana: `pip install "dstack-sdk[sol]"`

---

## Go SDK

**Install:**
```bash
go get github.com/Dstack-TEE/dstack/sdk/go
```

---
