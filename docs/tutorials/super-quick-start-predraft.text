### Quick Setup (30 seconds)

1. **Visit**: [dstack.dev/deploy](https://dstack.dev/deploy) 
2. **Sign in**: Use GitHub or email (instant signup)
3. **Select demo**: Choose one of the apps above

### Deploy Process (2 minutes)

<Accordion title="üñ±Ô∏è Web Deployment Steps">

**Step 2.1**: Click "Deploy Demo App"
- Choose your selected demo from Step 1
- dstack automatically pulls the container
- No building, no Docker commands needed

**Step 2.2**: Configure (optional)
- App name: `my-secure-demo`
- Resources: Auto-selected for demo
- Region: Closest to you

**Step 2.3**: Launch to TEE
- Click "Deploy to Secure Environment"
- Watch real-time deployment progress
- Get your secure HTTPS endpoint

**Result**: Live app running in hardware-protected TEE with cryptographic proof

### Security Verification (2 minutes)

<Accordion title="üîç Check TEE Attestation">

Your deployed app automatically includes these verification endpoints:

**Basic Security Check**:
```
Visit: https://your-app-url/security
See: Real-time security status and TEE information
```

**Cryptographic Proof**:
```
Visit: https://your-app-url/attestation
Get: Intel TDX attestation quote (cryptographic proof)
Verify: Upload to https://ra-quote-explorer.vercel.app/
```

**What This Proves**:
- Your app runs in genuine Intel TDX hardware
- Code hasn't been tampered with
- Memory is encrypted and isolated
- Independent verification possible

</Accordion>
