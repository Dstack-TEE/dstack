# The Recommended DStack Learning Path

### Beginner Path

1. Read [What is dstack?](/docs/overview/what-is-dstack)
2. Try the [Super Quick Start Guide](/docs/tutorials/super-quick-start) for instant TEE deployment (no setup required)
3. Explore [Core Concepts](/docs/concepts/overview)

### Advanced Path

1. Complete [Installation](/docs/getting-started/installation)
2. Study [System Architecture](/docs/concepts/architecture)
3. Review [Security & Research Overview](/docs/security-research/overview)
4. Deploy your own dstack:  
   - [Install dstack](/docs/getting-started/installation)  
   - [First Deployment](/docs/getting-started/first-deployment)
5. Once your dstack system is running, try the [Rust Tutorial](/docs/tutorials/rust-example) and [Python Tutorial](/docs/tutorials/python-example) on your own infrastructure

**Ready to start?** Begin with our [Super Quick Start Guide](/docs/tutorials/super-quick-start) →
