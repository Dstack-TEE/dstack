## What You'll Learn

dstack is a comprehensive platform designed to simplify the development, deployment, and management of applications. Whether you're an individual developer or part of a large team, dstack provides the tools and infrastructure you need to be productive.

## What is dstack?

dstack is a platform that helps you:

- **Build** applications with modern development workflows
- **Deploy** your applications to various environments
- **Manage** your applications throughout their lifecycle

## Why Choose dstack?

dstack stands out by making secure computing accessible:

- **Zero Code Changes**: Deploy existing Docker containers without modification
- **Hardware Agnostic**: Not tied to specific TEE implementations
- **Developer Friendly**: Use familiar tools and workflows
- **Enterprise Ready**: Production-grade security and compliance
- **Open Source**: Transparent, auditable, and community-driven
